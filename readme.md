raw链：https://raw.githubusercontent.com/Reindex-9/Clash-Custom/main/custom-rules.ini

#### 使用方法
1. “ACL4SSR在线订阅转换”
2. 点击“进阶模式”
3. 导入订阅链接（机场、自建的）
4. 远程配置把这个仓库里面的“.ini”结尾的文件地址填上去
5. 转换导入clash即可
